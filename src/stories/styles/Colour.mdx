<h1>Colour</h1>

<p>Always use the NSDS colour palette.</p>

<h2>Colour contrast</h2>
<p>
      Make sure the contrast ratio of text and interactive elements in your project meets <a style={{ color: '#2E5A60', textDecoration: 'underline' }} href="https://www.w3.org/TR/WCAG21/#contrast-minimum">level AA of the Web Content Accessibility Guidelines (WCAG 2.1).</a><br />
</p>

<h2>Main colours</h2>
<p>If you are using astrea-react-ds please use the provided variables, rather than copying the hexadecimal (hex) color values. For example, using $astrea-brand-color instead of #1d70b8, will allow your service to always use the most recent <a style={{ color: '#2E5A60', textDecoration: 'underline' }} >color palette</a> when updating.</p>

<p>Only use the variables in the context they’re designed for. In other cases, you should reference the colour palette directly. To use red, you should use nsds-color ("red") rather than $ astrea-error-color.</p>

<p>Only use this color for error messages. For example, when a user omits a mandatory field in a form.</p>

<p>Only use this colour for success messages. For example, use it as positive feedback when a user completes a task.</p>
