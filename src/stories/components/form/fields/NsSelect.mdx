<h1>Select</h1>

<h3>What is</h3>

  The Select component is a form input that allows users to choose an option
  from a dropdown list.

<h3>How it works</h3>

  The Select component displays a dropdown list containing selectable options.
  Users can click the dropdown arrow and choose a single option from the list.

<Primary />

<h3>Best Practices</h3>
<ul>
  <li>
    Limit the number of options in the dropdown list to improve user experience.
  </li>
  <li>
    Provide a default or placeholder option to prompt users to make a selection.
  </li>
  <li>
    Keep the dropdown list visible and accessible, especially on smaller screens
    or mobile devices.
  </li>
</ul>

<h3>Props</h3>
<Controls />

<h3>Accessibility</h3>
<ul>
  <li>Ensure the Select component is navigable via keyboard controls, allowing users to access options without a mouse.</li>
  <li>Associate the Select element with a corresponding label for screen readers by using proper HTML markup (`<label>` and `for` attribute).</li>
</ul>

<h3>Do</h3>
<ul>
  <li>
    Use the Select component when users need to choose one option from a list of
    available choices.
  </li>
  <li>
    Provide a clear label or placeholder text to indicate the purpose of the
    dropdown.
  </li>
</ul>

<h3>Don'ts</h3>
<ul>
  <li>
    Avoid using a Select component for a small number of static options where
    radio buttons or checkboxes might be more suitable.
  </li>
</ul>
